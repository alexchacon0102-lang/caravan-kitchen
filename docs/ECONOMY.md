## Curva de monetización natural

```
Día 1–3:   Solo monedas, loop aprendido, primeras ventas
Día 4–7:   Primera mejora, motivación de zona nueva
Día 7–14:  Segundo mapa, recetas raras, primer rewarded ad
Día 14–30: Tercera zona, bestiario parcial, primer IAP opcional
Día 30+:   Meta de colección, eventos de temporada, pase opcional
```

## Balance de economía (guidelines para diseño)

- El jugador siempre debe poder hacer al menos 1 mejora por cada 3–4 sesiones normales.
- Nunca bloquear progreso por falta de monedas si el jugador juega regularmente.
- Las mejoras caras deben siempre sentirse como logro, no como barrera.
- Los cristales nunca deben ser necesarios para la progresión central.
